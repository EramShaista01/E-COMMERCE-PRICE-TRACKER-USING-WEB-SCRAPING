# E-COMMERCE-PRICE-TRACKER-USING-WEB-SCRAPING
The Pricewise project is a Next.js and Tailwind CSS web app that tracks Amazon product prices. It uses Axios and Cheerio to scrape data, storing it in MongoDB via Mongoose. Serverless functions power logic, while Nodemailer sends email alerts for price changes. Built with TypeScript, it features an intuitive UI and a dynamic image carousel.
